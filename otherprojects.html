  1. Building automation & Security using CAN bus and IoT (M.Tech Thesis Project) 

This project aims to provide a common interface for control and security of home/building to the user at lower cost in terms of deployment, operation and easy to use interface for configuring the system after deployment. The robustness of CAN bus and the features offered by IOT is explored and applied here 

 
  1. DeepSense – A touch sensor made out of clothes 

In this project, we created a 3D touch sensor, made completely from fabrics. The motivation for this project is the vision for future wearable tech and consumer electronic products like mobile phones, laptops etc that can be washed, dried and reused. This project was carried out at MIT media lab – Welspun Smart Textiles workshop held at PDPU, Ahmedabad, Gujarat. This work was carried out independently. Google publicly announced about their project ‘jacquard’ later which is conceptually the same.

 
  1. PoC to convert existing electrical meters to prepaid

A PoC was built to collect data from electrical meters over DLMS protocol via optical port. The data collected was uploaded to cloud for UI and also listen to instructions for connect/disconnect. This was done on STM32 HW and FreeRTOS.

 
  1.  Li-Ion BMS data logger and safety control (hobby project)

Created a PoC to collect data from Li-Ion Battery management system over CAN, upload it to cloud via MQTT and store in SD card in case of communication failure. The implementation was done on STM32 HW and Zephyr RTOS

  1.  Custom Remote Power Measurement & Management (M.Tech Mini Project) 

This project aims at providing the user control over his electric power consumption at home/office from a remote location. The system is designed using a modular approach so that user can have flexibility of choosing the facilities offered by the system. 

 
  1.  Kitchen Automation(Hobby Project)

Food cooking automation for domestic kitchen based on concept of IoT. This is being implemented on BeagleBone black SBC & CC3200. As of now the user can initiate boiling of eggs, rice, legumes, pulses etc from a remote location. This was made possible by hacking existing induction cook top.

https://github.com/Rajeevah/Bajaj-Induction-Cooktophack/wiki

 
  1. Implementation of MQTT broker on ESP8266 & CC3200 for nodes on a small intranet

Created a MQTT broker on a small Wifi SoC so that nodes in a local intranet can exchange data over MQTT. This was later given for a startup for evaluation of their home automation project

 
  1. NIYANTRAN (BE main project – KSCST (State Government funded project)) 

The project aimed at making a working model of a system which can be used by quadriplegic people for controlling the computer system and thus earning their livelihood out of it. This was implemented on 8 bit AVR and also 32 bit ARM controllers with extended features. As a demo, we tested its use at a local customer care.  http://www.kscst.iisc.ernet.in/sppArchive/public/describe/project/033/6220
